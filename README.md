# Doc Engine

A custom document rendering engine written in Node.js.

## Features (v0.1)

- Extracts DOCX structure
- Parses Word XML
- Builds internal document model (AST)
- Simple layout engine
- Generates basic PDF output

## Roadmap

- [ ] Text styling (bold, italic)
- [ ] Line wrapping
- [ ] Multi-page support
- [ ] Tables
- [ ] Images

## Goal

To build a fully controlled document rendering engine without relying on external conversion services.
